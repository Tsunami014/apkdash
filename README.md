# apkdash
A TUI dashboard for editing apk files.

This is purely for educational purposes to see how apk files work, do not use it on things you don't own, I do not take any responsibility for this program's use or modification.

## Running
Ensure you have:
- An apk file of an app that you have permission from the owner that you can look into it
```bash
pip install requests readchar
```

Then run `python3 main.py`
